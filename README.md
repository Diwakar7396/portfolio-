# Diwakar B — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Deploy on GitHub Pages

1. Create a new repository named `portfolio` on GitHub (or any name you prefer).
2. Upload all files **keeping the folder structure intact**:
   ```
   index.html
   css/style.css
   js/main.js
   Diwakar_B_Resume.pdf
   README.md
   ```
3. Go to **Settings → Pages → Source** → select `main` branch → root (`/`) → Save.
4. Your site will be live at: `https://Diwakar7396.github.io/portfolio/`

## 📁 File Structure
```
portfolio/
├── index.html            ← Main page (GitHub Pages entry point)
├── css/
│   └── style.css         ← All styles
├── js/
│   └── main.js           ← All interactions & animations
├── Diwakar_B_Resume.pdf  ← Downloadable resume
└── README.md             ← This file
```

## ✨ Features
- Animated loader / welcome screen
- Typewriter hero text
- Scroll-reveal animations
- Back-button goodbye overlay
- Contact form via FormSubmit (no backend needed)
- Fully responsive: mobile, tablet, desktop
- iOS safe-area support
- Keyboard accessible & reduced-motion friendly

The form uses [FormSubmit.co](https://formsubmit.co) — no server required.
